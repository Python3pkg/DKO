public static class Condition.Literal
extends Condition

Use this class to add custom SQL code to your where clause.   For Example: 

   SomeClass.ALL.where(new Condition.Literal("id = 1"));
Obviously this should be used sparingly, and can break compatibility between different databases.   But it's here if you really need it.
