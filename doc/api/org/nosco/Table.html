org.nosco
Class Table

java.lang.Object
  org.nosco.Table

public abstract class Table
extends java.lang.Object

This is the base class of all classes generated by this API. Some of these methods are public only by necessity. Please use only insert(), update(), save() and dirty(). I consider all others fair game for changing in later versions of the API.
