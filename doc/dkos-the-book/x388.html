4.10. Selecting Only Specific Fields

DKOs support limiting the fields selected in a query. Usually this is unnecessary as the DKO library will watch how an object is used and automatically stop selecting unused fields, but sometimes it is useful to be explicit. Here is an example:

System.out.println(Patient.ALL.first().toStringDetailed());
System.out.println(Patient.ALL.onlyFields(Patient.LAST_NAME).first().toStringDetailed());

==> select p.social_security_number, p.birth_date, p.last_name, p.first_name, p.id, p.since, p.primary_doctor_id from patient p  limit 1
[Patient id:1 social_security_number:null birth_date:null last_name:Anderson first_name:Derek since:null primary_doctor_id:null]
==> select p.last_name from patient p  limit 1
[Patient last_name:Anderson]
